## ğŸš€ Key Features & Report Pages

This report contains **2 main pages** (Overview and Performance Notes) designed to track core retail KPIs and performance across different product brands and regions:

* **Executive Overview:** Displays key current month KPIs (Transactions: **18.3K**, Profit: **$71.6K**, Returns: **496**) and weekly revenue trends.
* **Brand Performance:** Detailed matrix showing total sales, profit margin, and return rate per product brand.
* **Geospatial Analysis:** Visualization of transactions and returns across **USA, Canada, and Mexico**.
* **Target Tracking:** Revenue vs. Target gauge visualization to assess performance against monthly goals (e.g., **$120K**).
